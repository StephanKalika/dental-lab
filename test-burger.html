Тест бургер-меню

Цей файл створено для тестування бургер-меню на мобільних пристроях.

Інструкції для тестування:

  1. Відкрийте цю сторінку в мобільному браузері
  2. Спробуйте натиснути на бургер-кнопку (три лінії) у верхньому правому куті
  3. Перевірте, чи змінюється анімація кнопки
  4. Дивіться на статус внизу екрану

Очікуваний результат: При натисканні кнопка має змінити анімацію (три лінії перетворюються на хрестик)
